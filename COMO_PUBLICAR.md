# Como colocar no ar (passo-a-passo Débora)

## Passo 1 — Criar conta GitHub (se ainda não tiver)
- Acesse https://github.com/signup
- Use email da Mitawa (ex: jurídico@mitawa... ou pessoal mesmo)
- Confirme o email

## Passo 2 — Baixar os arquivos do app
- Os 4 arquivos que recebeu:
  - `index.html`
  - `README.md`
  - `.nojekyll`
  - `.gitignore`
- Coloque todos numa pasta `cachorro-do-bigode-desempenho` no seu computador

## Passo 3 — Criar o repositório
1. Logada no GitHub, clique no `+` no canto superior direito → **New repository**
2. **Repository name**: `cachorro-do-bigode-desempenho`
3. Marque **Public** (necessário pro Pages funcionar gratuito)
4. **NÃO** marque "Add a README" (a gente já tem)
5. Clique **Create repository**

## Passo 4 — Subir os arquivos
1. Na página do repo recém-criado, clique **uploading an existing file** (link azul)
2. Arraste os 4 arquivos da pasta para a área de upload
3. Em **Commit changes**, escreva: `primeira versão`
4. Clique **Commit changes**

## Passo 5 — Ativar GitHub Pages
1. No repo, clique aba **Settings** (engrenagem, no topo)
2. No menu lateral esquerdo, clique **Pages**
3. Em **Source**:
   - Branch: `main`
   - Folder: `/ (root)`
4. Clique **Save**
5. Aguarde 1–2 minutos. Atualize a página.
6. Aparecerá: `Your site is live at https://SEUUSUARIO.github.io/cachorro-do-bigode-desempenho/`

## Passo 6 — Usar no iPhone
1. Abra o link no Safari do iPhone
2. Toque no ícone de compartilhar (□ com seta pra cima)
3. Role e toque em **Adicionar à Tela de Início**
4. Vira ícone igual app. Funciona offline depois da primeira abertura.

## Pronto!

> ⚠️ **IMPORTANTE**: como o app salva no navegador local, cada gestor terá seus próprios dados se acessar de dispositivos diferentes. Para v2 (compartilhado em tempo real entre toda a rede), precisa evoluir pra backend — me avisa quando quiser.
